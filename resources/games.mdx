Unblocked game sites are web-based game collections hosted on domains that school content filters haven't caught up with yet. Because blocklists are maintained manually or updated infrequently, new mirror domains often stay accessible for weeks or months before being flagged. Services like **now.gg** take a different approach — they stream Android games from remote servers, so the Chromebook itself only runs a browser tab, bypassing app restrictions entirely. Browser-based emulators let you upload a ROM file downloaded from a ROM site and play it directly in the tab using WebAssembly or JavaScript CPU emulation, requiring no installation or developer mode.

## Emulators

Browser-based emulators let you play ROMs from any console directly in the tab. You need to download a ROM file first, then upload it to the emulator.

### Emulator Sites

These emulators have no pre-loaded games — you supply your own ROM:

- [binbashbanana.github.io/webretro/](https://binbashbanana.github.io/webretro/) — WebRetro (supports many consoles)
- [demo.emulatorjs.org](https://demo.emulatorjs.org/) — EmulatorJS demo

**GBA-focused (EmulatorJS + built-in games):**
- [cattn.github.io/gba/](https://cattn.github.io/gba/) — v3.0
- [mathstudy.dev/gba/](https://mathstudy.dev/gba/) — v2.0

### ROM Download Sites

Download ROM files to use with the emulators above:

- [romspedia.com](https://www.romspedia.com/)
- [theromdepot.com](https://theromdepot.com) — free account, no email required
- [romsdownload.net](https://romsdownload.net/)
- [hexrom.com](https://hexrom.com/)

### How to Play Console Games

<Steps>
  <Step title="Download a ROM">
    Visit a ROM site above (e.g. `romspedia.com`) and find the game you want. Download the ROM file to your Chromebook's Downloads folder.
  </Step>
  <Step title="Open an emulator">
    Go to one of the browser emulator sites above, such as `binbashbanana.github.io/webretro/` or `demo.emulatorjs.org`.
  </Step>
  <Step title="Select your console">
    Find the console the game runs on (NES, SNES, GBA, PS1, etc.) or click **Auto-detect** if available.
  </Step>
  <Step title="Upload your ROM">
    Click **Upload File**, navigate to your Downloads folder, and select the ROM file.
  </Step>
  <Step title="Play">
    Click Done or Start. The game will launch in-browser. Many GBA sites also have pre-loaded games so you can skip the download step entirely.
  </Step>
</Steps>

<Note>
  Game site domains rotate frequently as school filters update. If a link is dead, check for a newer mirror in the project's GitHub repository or the relevant Discord server.
</Note>
