"Mixes" (also called Prxes) are a step up from plain proxy tools. Where a raw proxy like Rammerhead only provides a tunnel to browse the web, a Mix site bundles that proxy functionality together with a built-in game library, an app launcher for popular services, and sometimes a custom UI. Visiting a Mix site feels more like visiting a game portal than a proxy — which also helps them stay under the radar of school administrators scanning for proxy-looking URLs. The sites listed here span a wide range of projects, each maintained by different communities with their own Discord servers.

<Note>
  The Mixes folder of this repository is **unmaintained**. Many links in this section are dead or intermittently available. For the most current working instances, check the Discord server listed under each project. The combined link list and master directory below reflect the full contents of the source files at the time of collection.
</Note>
