Web proxies are server-side tools that route your browser's traffic through an intermediary URL, letting you reach sites that would otherwise be blocked at the network level. Unlike a VPN — which encrypts all traffic from your device at the OS level — a web proxy works entirely inside the browser tab: you visit the proxy's URL, type in the destination, and the proxy fetches the page on your behalf. The four proxy flavors covered here are **Rammerhead** (a full session-based proxy that can be cloaked and opened as `about:blank`), **Hypertabs** (simple hosted instances on surge.sh), **Scramjet** (a newer service-worker proxy by MercuryWorkshop), and **Page Sandbox** (Cloudflare Workers–based instances).

<Note>
  The `Prxes` folder of this repository is **unmaintained**. Links go dead regularly as domain registrars reclaim free TLDs. For fresher links, check the [Mixes](/resources/mixes) section or join the **Titanium Network Discord** at [discord.gg/unblock](https://discord.gg/unblock).
</Note>

## Scramjet

Scramjet is a newer open-source proxy built by MercuryWorkshop that uses service workers to intercept and rewrite requests in the browser. It supports a wider range of sites than older proxies.

```
https://scramjet.mercurywork.shop/
```

Source code and self-hosting instructions: [github.com/MercuryWorkshop/scramjet](https://github.com/MercuryWorkshop/scramjet)

## How to Check if a Proxy is Working

Rather than opening each link one by one to test it, use a multi-URL opener to batch-check a list of proxies all at once.

<Steps>
  <Step title="Copy the proxy URLs you want to test">
    Grab a set of links from the lists above and paste them into a text editor.
  </Step>
  <Step title="Open a multi-URL tool">
    Navigate to one of the following services:
    - [openallurls.com](https://www.openallurls.com/)
    - [openmultipleurl.com](https://www.openmultipleurl.com/)
    - [websiteplanet.com/webtools/multiple-url](https://www.websiteplanet.com/webtools/multiple-url/)
  </Step>
  <Step title="Paste the URLs and open them">
    Paste your list of proxy URLs into the input box and click the button to open all of them. Your browser will open each in a new tab.
  </Step>
  <Step title="Check which tabs loaded">
    Any tab that loads the Rammerhead (or other proxy) interface is working. Close the ones that show errors, and bookmark the working ones.
  </Step>
</Steps>

## Troubleshooting

<Accordion title="All of the proxies in the repo are blocked on my school network">
  Join a Discord server like the Titanium Network Discord to get fresh links from community members who update them constantly. The repository is unmaintained, but community Discord servers stay up to date.
</Accordion>

<Accordion title="Discord is blocked on my school Chromebook">
  Use another device — your phone, a home computer, or a library computer — to access Discord and find updated proxy links.
</Accordion>

<Accordion title="The word 'proxy' is filtered by my school's content filter">
  Try searching in a different language. Searching **"прокси"** (Russian for proxy) or a translation in another language has worked around keyword-based filters. Alternatively, search on your personal phone or a device not connected to the school network.
</Accordion>

<Accordion title="I need a proxy but can't access Discord or GitHub to find one">
  Search **"School Chromebook proxy 2026"** on YouTube from a personal device. YouTube tutorials regularly share working links in descriptions and comment sections.
</Accordion>

<Accordion title="I don't have a GitHub account to access repository links">
  Create a free GitHub account at [github.com/signup](https://github.com/signup). If the signup page is blocked at school, create the account from your phone or home computer.
</Accordion>
